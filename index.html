Angular DevTools

A Chrome Extension for Debugging Angular Apps

Lightning Talk

What Is Angular DevTools?

  • Free Chrome & Firefox extension built by the Angular team
  • Adds an "Angular" tab to your browser DevTools
  • Works with Angular 9+ apps running in development mode
  • Replaces the older Augury extension

Standard browser DevTools can inspect the DOM, but they don't understand Angular's component model. Angular DevTools bridges that gap.

Getting Started

1 Search "Angular DevTools" in the Chrome Web Store
2 Click Add to Chrome
3 Open DevTools on any Angular app (F12 or Ctrl+Shift+I)
4 Click the new Angular tab

Note: Only works with development builds. Production builds strip the debug hooks Angular DevTools needs.

Three Main Features

Components

Browse the full component tree. Select any component to inspect its inputs, outputs, and current state in real time.

Profiler

Record change detection cycles. See which components re-rendered, what triggered it, and how long each cycle took.

Injector Tree

Visualize the dependency injection hierarchy. Understand how and where services are provided and resolved.

Component Explorer

  • Shows the full component hierarchy as a tree, not the DOM
  • Click any component to see its properties and their current values
  • Edit values live to test different states without changing code
  • Navigate to the component's source file directly from the panel

Component Explorer in Action

Selecting app-user-detail reveals its properties, injected services, and live state.

Profiler

  • Hit Record, interact with your app, then Stop
  • See each change detection cycle as a frame with timing data
  • Identify which components are re-rendering unnecessarily
  • Spot performance bottlenecks before they hit production
